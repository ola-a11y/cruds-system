<h2>‚ú® Key Features </h2>

* **Create:** Add new products with details like price, taxes, ads, and discount.
* **Read:** Display all products in an organized table.
* **Update:** Edit the data of an existing product..
* **Delete:** Delete a single product or delete all products at once.
* **Search:** Find products by either <strong>Title</strong> or <strong>Category</strong>.
* **Auto-Calculate:** Automatically calculates the total price based on inputs.
* **Local Storage:** All data is saved in the browser's <code>LocalStorage</code>, so no data is lost on page reload.
